flagship solutions

smart iiot solution
현재 수집되고 있지 않은 PLC, CIM, File 데이터 수집을 위한 설비 개조, 변경 없이 데이터를 수집하여 분석에 필요한 Big-Data 구축과 Rule을 통한 설비 제어가 가능한 제조공정에 특화된 IoT System입니다.
RTMS / Pcap(Packet Capture) Service
㈜제이엠소프트는 10여년간 삼성디스플레이㈜의 각 생산 라인에 특화된 공정별 통합 모니터링 시스템 구축 및 유지보수를 진행했던 개발 인력들의 경험을 바탕으로 Line Engineering System 분야 및 대용량 고속처리 DB에 많은 노하우와 기술을 축적/보유하고 있습니다.
Remote Control System
RCS는 원격지에 있는 다양한 원격 장치를 원격으로 통합 관리, 모니터링 및 제어가 가능한 솔루션입니다. 원격지에 있는 Desktop PC, Server, IP Camera (CCTV), HMI, IoT Device, VNC Server, PC기반의 산업장비, 키오스크와 같은 무인 PC등을 관리자 PC에서 통합 관리, 모니터링, 원격 제어를 합니다.
A.I Machine & Deep Learning (아르고스)
머신러닝을 기반으로 데이터전처리, 데이터분석, 변수선정, 피쳐셀렉션, 인공지능모델생성, 모델평가, 모델시뮬레이션, 모델테스트결과를 통한 기계학습, 강화학습, 가상계측 기반의 MES, APC, R2R 솔루션입니다.
